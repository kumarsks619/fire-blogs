# FireBlogs

A blogging website built with VueJS + Vuex + Firebase.

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint
```

### Steps to Deploy

```
npm run build
firebase deploy
```
